## 🚀 Cómo publicarlo en GitHub Pages

1. **Crea un repositorio nuevo** en GitHub (público, para que Pages sea gratis).
2. **Sube TODOS los archivos de esta carpeta** a la raíz del repositorio (arrastra y suelta en la página de GitHub, o vía Git). Importante: todos juntos, en la raíz — no en subcarpetas.
3. Entra a **Settings → Pages** del repositorio.
4. En "Source", selecciona la rama `main` (o `master`) y carpeta `/ (root)`.
5. Guarda. GitHub te da un link tipo `https://tu-usuario.github.io/tu-repo/` — en unos minutos el sitio queda publicado ahí.
6. Como el archivo se llama `index.html`, GitHub Pages lo carga automáticamente como página principal.
